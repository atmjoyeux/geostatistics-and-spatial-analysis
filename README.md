# Geostatistics and Spatial Analysis Scripts

This repository contains a collection of R and Python scripts for geostatistical and spatial data analysis.

## Contents

- Multivariate analysis (PCA, NMDS, PERMANOVA, Mantel tests)
- Spatial autocorrelation (Moran’s I)
- Granulometry and sediment analysis
- Data visualization (dendrograms, polar plots)
- Profile generation (Python)

## Languages

- R
- Python

## Notes

These scripts were developed as part of research workflows in environmental data analysis.
